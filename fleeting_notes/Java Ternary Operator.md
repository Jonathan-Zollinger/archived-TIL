
#cheatsheet #java

#### Syntax:

```java
variable = expression1 ? expression2: expression3
```

#### Example:


```java
num1 = 10;
num2 = 20;

res = ( num1 > num2 ) ? ( num1 + num2 ): ( num1 - num2 )

// Since num1<num2, 
// the second operation is performed
res = num1-num2 = -10
```
